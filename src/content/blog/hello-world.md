# Hello, world

This is the first post on my new personal website. I built it to **showcase the
projects I'm working on** and to write short notes about what I'm learning.

## What you'll find here

- **Projects** — the things I build, with videos and write-ups
- **Blog** — learning notes and weekly updates
- **About** — a bit more about me and how to get in touch

## How I write these posts

Each post is just a Markdown file in `src/content/blog/`. I can use the usual
formatting:

- **bold** and *italic* text
- [links](https://example.com)
- lists like this one
- `inline code` and code blocks:

```js
console.log('Hello from a blog post!')
```

> Quotes work too — handy for highlighting something.
